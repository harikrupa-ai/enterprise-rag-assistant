# Enterprise RAG Assistant

A Streamlit-based PDF Question Answering application built with Python, Streamlit, and PyPDF.

## Features

- Upload PDF files
- Extract text from PDFs
- Ask questions about document content
- Simple document search

## Tech Stack

- Python
- Streamlit
- PyPDF

## Run Locally

pip install streamlit pypdf

streamlit run main.py
